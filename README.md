# news_scraping

Yahooニュース自動収集ツール

■ 概要

Yahooニュースの最新見出しとURLを自動で取得し、CSVファイルとして保存するツールです。
毎日決まった時刻に自動実行されるため、ニュース収集作業を完全に自動化できます。

-----------------------------------------------------------------------

■ 主な機能
•	指定時刻に自動でデータ取得
•	最新ニュースの見出し・URLを収集
•	CSV形式でデータ出力
•	重複データの自動排除
•	負荷を抑えたスリープ制御（効率的な監視処理）

-----------------------------------------------------------------------

■ 使用技術
•	Python
•	requests
•	BeautifulSoup（bs4）
•	csv
•	datetime
•	time

-----------------------------------------------------------------------

■ 使用方法
1.	ターミナルを開く
2.	プログラムを実行
3.	news_日付.csv が生成される
4.	CSVを開いてデータを確認

※プログラムを起動したままにすると、毎日自動で更新されます。

■ カスタマイズ方法
•	取得件数の変更
→ コード内のスライス部分を変更
•	取得対象サイトの変更
→ URLとパース処理を書き換える
•	実行時間の変更
→ auto_job(hour, minute, second) を変更
